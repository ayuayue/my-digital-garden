# 表格

安装了插件  Advanced Tables。使用时只需要输入 | 再输入表头，然后使用 tab 键即可自动补全。

如：

| 姓名 | 性别 | 爱好 | 
| ---- | ---- | ---- |

# Task

输入一个 todolist 类似的东西。使用 - [ ] 表示未完成的，- [x] 表示已完成。也可以使用 Task 插件，使用命令  create or edit task 来添加一个 task。

如：

- [ ] 博客发布 
- [x] 完成 ✅ 2023-05-23
